# 主要登場人物データベース

本ファイルは、物語に登場する主要な脇役キャラクター（主人公候補を含む）のデータベースです。
「未知への探求心」を持つ人物を中心に、この世界の多様性と奥行きを表現します。

────────────────────────────────────────

## 【未知を探求する者たち】

この世界では「集落内に籠って生活する」のが一般的であり、未知を解き明かしたい・冒険したいと考える者は極めて稀です。しかし、そのような人物は実利的な動機（集落や人々のため）を持ち、変わり者として見られながらも、時折その知識が命を救います。

────────────────────────────────────────

### 1. ルイン（地図屋）

**基本情報**:
- 年齢：30代女性
- 出身：南部の小規模集落「草原の村」（元々）
- 現在地：交易中継集落「三叉路の町」（人口1,800人）
- 職業：元隊商（15年）→ 現在は地図製作者
[![ルイン](URL)](URL)
**外見**:
- 日焼けした肌、がっしりした体格
- 短く刈った髪（実用性重視）
- 隊商時代の傷跡が腕や脚にある
- 鋭い観察眼を持つ茶色の瞳

**性格**:
- 実務的で感情を表に出さない
- 「命を救う」ことへの強い使命感
- 冒険心よりも「仲間を守りたい」という動機
- 無駄な言葉を嫌う

**背景**:
- 隊商として15年間、各地を巡った
- 移動中に見た地形や危険生物の情報を記録する癖があった
- 親友だった隊商仲間を道中の事故で失った経験がトラウマ
- 「正確な地図があればあの事故は防げた」という後悔が原動力
- 引退後、地図製作を本格的に始める

**現在の活動**:
- 三叉路の町を拠点に、隊商から情報を集めて地図を更新
- 自ら短期間（5-10日）の調査旅行に出る
- 危険な場所の「迂回ルート」を発見することが目標
- 地図を隊商に売る（50-200穀/枚、ルートの危険度で変動）
- 年間収入：約1,500-2,000穀（十分な生活費）

**口癖・名言**:
「私は冒険がしたいわけじゃない。隊商仲間が死ぬのを見たくないだけよ」
「地図は命を救う。それ以上でも、それ以下でもない」

**周囲の評価**:
- 「変わり者だが、彼女の地図は命を救う」
- 実利的な価値があるため尊重される
- ただし「そこまでする必要があるのか」という疑問の目も
- 隊商からは絶大な信頼を得ている

**能力**:
- 地形の記憶力が異常に高い（一度通った道は忘れない）
- 危険生物の痕跡から種類と数を推測できる
- 隊商としての戦闘技術（護衛レベル）
- 簡易な測量技術（歩測、目測）

**物語での役割**:
- 現地知識と経験による地図製作
- 最も協力的な仲間になりうる
- 新しいルートの発見に貢献

**関係性**:
- 弟子：若い隊商ナイル（20代男性）に地図の読み方を教えている
- 協力者：三叉路の町の長老会議と良好な関係
- ライバル：他の地図屋はいないが、一部の隊商は「勘で十分」と彼女を軽視

### 2. ダルガン（獣師）

**基本情報**:
- 年齢：40代男性
- 出身：辺境の小規模集落「北嶺村」（人口450人）
- 現在地：北嶺村（拠点）、安全圏境界付近の観察小屋
- 職業：元狩人（エリート）→ 現在は危険生物研究者

**外見**:
- がっちりした筋肉質の体格
- 無数の傷跡（獣との戦いの証）
- 鋭い目つき、常に周囲を警戒している
- 獣の臭いが染み付いた革製の服

**性格**:
- 寡黙で内省的
- 観察眼が鋭く、細部にこだわる
- 「理解すること」への執念
- 一見、狂気じみているが論理的

**背景**:
- 狩人集団のエリートとして20年以上活躍
- ある日「なぜこの獣はこう行動するのか」という疑問に囚われる
- 集落を守るには「獣を殺す」だけでなく「獣を知る」必要があると確信
- 安全圏境界付近に観察小屋を建て、危険生物を観察し始める

**現在の活動**:
- 観察小屋で危険生物の行動パターンを記録
- 危険生物の「縄張り」「繁殖期」「移動パターン」を研究
- 獣の骨や毛皮から「どの獣が近くにいるか」を判別する技術を開発
- 年に数回、他集落を訪れて知識を共有（報酬：100-300穀/回）
- 弟子を2人取っており、知識を継承しようとしている

**口癖・名言**:
「獣を殺すだけじゃ終わらない。なぜそこにいるのか、なぜ増えるのか。それを知らなければ、いつか集落は滅ぶ」
「獣は敵じゃない。理解すれば、避けられる」

**周囲の評価**:
- 「頭がおかしくなった」と陰で言われる
- しかし彼の知識で集落が何度も救われている
- 狩人たちは彼の助言を重視する
- 子どもたちは彼を恐れている

**能力**:
- 危険生物の行動予測（繁殖期、移動ルート）
- 痕跡から獣の種類・数・状態を判別
- 狩人としての戦闘技術（最高レベル）
- 忍耐力（何日も観察小屋に籠もれる）

**物語での役割**:
- 現場の知恵による生態系の理解
- 危機への対策を発見
- 生態系の変化を物語の重要な要素として描く

**関係性**:
- 弟子：リク（18歳男性）とセナ（16歳女性）
- 妻：エルダ（集落に残っている、夫の仕事を理解しつつ心配）
- 協力者：近隣集落の狩人たちと情報交換

### 3. セリカ（古代を探る者）

**基本情報**:
- 年齢：20代女性
- 出身：都市遺跡周辺集落「旧都の門」（人口2,100人）
- 現在地：旧都の門
- 職業：研究職（記録係の補佐）

**外見**:
- 細身で華奢な体格
- 眼鏡をかけている（稀少品、ガラスレンズ）
- インクで汚れた指
- 好奇心に満ちた瞳

**性格**:
- 知的好奇心が旺盛
- 理想主義的で頑固
- 集落の保守的な空気に不満
- 「祖先の知識を取り戻す」という使命感

**背景**:
- 集落の「記録係」の娘として育つ
- 幼い頃から古代の遺物に囲まれて育った
- 「大災害前の技術を取り戻せば、もっと豊かに暮らせる」という信念
- 長老たちは「余計なことをするな」と彼女を抑制しようとする

**現在の活動**:
- 廃墟となった旧都市の遺跡を探索（週に1-2回、護衛付き）
- 古代の道具や文字（読めないが）を収集
- 隊商に依頼して、他の遺跡からの情報を集める
- 集落の長老会議に「古代技術の復元」を提案（却下されることが多い）
- 時折、彼女の発見が集落に利益をもたらす（新しい作物の栽培法など）

**知識の獲得経路**:
- 父エリウス（記録係）から継承した断片的な古代知識
- 集落に保管されている古文書や遺物への幼少期からのアクセス
- 隊商から購入した他の遺跡の断片文字との比較研究
- 試行錯誤による推測（完全な解読ではなく、パターン認識による推測）
- 実用的な知識（道具の使い方など）は実験と観察から習得

**口癖・名言**:
「私たちは祖先より貧しく生きている。それは間違っている」
「知識を失うことは、死ぬことより恐ろしい」

**周囲の評価**:
- 「理想主義の若者」と冷ややかに見られる
- しかし実績もあるため完全には否定されない
- 一部の若者は彼女に共感している
- 長老たちは「危険な思想」として警戒

**能力**:
- 古代文字の解読（一部のみ、試行錯誤）
- 遺物の機能推測（観察力と推理力）
- 記録・整理能力（几帳面）
- 説得力のある文章を書く

**物語での役割**:
- 古代知識による重要な発見
- 真実を知ったときの葛藤（知識を公開すべきか、隠すべきか）

**関係性**:
- 父：記録係のエリウス（娘の探求心を理解しつつ心配）
- 協力者：若い護衛トーマス（遺跡探索に同行）
- 対立者：保守派の長老たち

- 尊重はされるが、完全には理解されない
- 「そこまでする必要があるのか」という疑問の目

### 3. 孤独な戦い
- 理解者は少ない
- しかし時折、彼らの知識が命を救う

### 4. 物語への貢献
- 世界の多様性と奥行きを表現
- 読者の共感を得やすい人間的なキャラクター
- 異なる視点を提供

## 【外部からの来訪者：調査隊生存者】

約15年前（ヤヒロ内部時間）、因果の泡に侵入した調査隊の生存者たち。元々25名の調査隊だったが、境界通過時に10名が死亡、その後15年間で3名が事故・病気で死亡。現在12名が生存している。

彼らは「帰還」「共生」「救済」という3つの理念に分かれ、それぞれ異なる方法でヤヒロと関わっている。全員が善意を持ち、それぞれの正義に基づいて行動している。

### 調査隊のタイムライン

**フロディアス出発→ヤヒロ到達→現在**:
1. **フロディアス出発**: 各メンバーの出発時年齢（下記参照）
2. **航行期間**: コールドスリープで客観100年、主観1年
3. **ヤヒロ外部での準備期間**: 2年（ナノマシンによる老化抑制60%適用 = 実質1.2歳加齢）
4. **因果の泡侵入**: 15年前（ヤヒロ内部時間）
5. **現在**: 侵入から15年経過（ナノマシン老化抑制60%適用 = 実質9歳加齢）

**年齢計算式**: 出発時年齢 + 1年（航行） + 1.2年（準備） + 9年（ヤヒロ15年） = 現在年齢

**長寿化の設定**:
- 移民星人類のDNA設計: 平均寿命90-100歳、老化速度70%
- ナノマシン投与者: 老化速度さらに60%に抑制、実効寿命150-200歳
- 見た目年齢: 実年齢より10-15歳若い

物語に登場させる際は、彼らの「変わり者」としての苦悩、周囲との軋轢、そして信念を貫く強さを描くことで、世界に深みと人間ドラマを加えることができます。
