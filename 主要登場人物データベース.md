# 主要登場人物データベース

本ファイルは、物語に登場する主要な脇役キャラクター（主人公候補を含む）のデータベースです。
コウキ以外の「未知への探求心」を持つ人物を中心に、この世界の多様性と奥行きを表現します。

────────────────────────────────────────

## 【未知を探求する者たち】

この世界では「集落内に籠って生活する」のが一般的であり、未知を解き明かしたい・冒険したいと考える者は極めて稀です。しかし、そのような人物は実利的な動機（集落や人々のため）を持ち、変わり者として見られながらも、時折その知識が命を救います。

────────────────────────────────────────

### 1. ルイン（地図屋）

**基本情報**:
- 年齢：30代女性
- 出身：南部の小規模集落「草原の村」（元々）
- 現在地：交易中継集落「三叉路の町」（人口1,800人）
- 職業：元隊商（15年）→ 現在は地図製作者

**外見**:
- 日焼けした肌、がっしりした体格
- 短く刈った髪（実用性重視）
- 隊商時代の傷跡が腕や脚にある
- 鋭い観察眼を持つ茶色の瞳

**性格**:
- 実務的で感情を表に出さない
- 「命を救う」ことへの強い使命感
- 冒険心よりも「仲間を守りたい」という動機
- 無駄な言葉を嫌う

**背景**:
- 隊商として15年間、各地を巡った
- 移動中に見た地形や危険生物の情報を記録する癖があった
- 親友だった隊商仲間を道中の事故で失った経験がトラウマ
- 「正確な地図があればあの事故は防げた」という後悔が原動力
- 引退後、地図製作を本格的に始める

**現在の活動**:
- 三叉路の町を拠点に、隊商から情報を集めて地図を更新
- 自ら短期間（5-10日）の調査旅行に出る
- 危険な場所の「迂回ルート」を発見することが目標
- 地図を隊商に売る（50-200穀/枚、ルートの危険度で変動）
- 年間収入：約1,500-2,000穀（十分な生活費）

**口癖・名言**:
「私は冒険がしたいわけじゃない。隊商仲間が死ぬのを見たくないだけよ」
「地図は命を救う。それ以上でも、それ以下でもない」

**周囲の評価**:
- 「変わり者だが、彼女の地図は命を救う」
- 実利的な価値があるため尊重される
- ただし「そこまでする必要があるのか」という疑問の目も
- 隊商からは絶大な信頼を得ている

**能力**:
- 地形の記憶力が異常に高い（一度通った道は忘れない）
- 危険生物の痕跡から種類と数を推測できる
- 隊商としての戦闘技術（護衛レベル）
- 簡易な測量技術（歩測、目測）

**物語での役割**:
- コウキの広域監視網データと彼女の現地知識を組み合わせる展開
- 「技術（コウキ）vs 経験（ルイン）」の対比
- 最も協力的な仲間になりうる
- 新しいルートの発見に貢献

**関係性**:
- 弟子：若い隊商ナイル（20代男性）に地図の読み方を教えている
- 協力者：三叉路の町の長老会議と良好な関係
- ライバル：他の地図屋はいないが、一部の隊商は「勘で十分」と彼女を軽視

### 2. ダルガン（獣師）

**基本情報**:
- 年齢：40代男性
- 出身：辺境の小規模集落「北嶺村」（人口450人）
- 現在地：北嶺村（拠点）、安全圏境界付近の観察小屋
- 職業：元狩人（エリート）→ 現在は危険生物研究者

**外見**:
- がっちりした筋肉質の体格
- 無数の傷跡（獣との戦いの証）
- 鋭い目つき、常に周囲を警戒している
- 獣の臭いが染み付いた革製の服

**性格**:
- 寡黙で内省的
- 観察眼が鋭く、細部にこだわる
- 「理解すること」への執念
- 一見、狂気じみているが論理的

**背景**:
- 狩人集団のエリートとして20年以上活躍
- ある日「なぜこの獣はこう行動するのか」という疑問に囚われる
- 集落を守るには「獣を殺す」だけでなく「獣を知る」必要があると確信
- 安全圏境界付近に観察小屋を建て、危険生物を観察し始める

**現在の活動**:
- 観察小屋で危険生物の行動パターンを記録
- 危険生物の「縄張り」「繁殖期」「移動パターン」を研究
- 獣の骨や毛皮から「どの獣が近くにいるか」を判別する技術を開発
- 年に数回、他集落を訪れて知識を共有（報酬：100-300穀/回）
- 弟子を2人取っており、知識を継承しようとしている

**口癖・名言**:
「獣を殺すだけじゃ終わらない。なぜそこにいるのか、なぜ増えるのか。それを知らなければ、いつか集落は滅ぶ」
「獣は敵じゃない。理解すれば、避けられる」

**周囲の評価**:
- 「頭がおかしくなった」と陰で言われる
- しかし彼の知識で集落が何度も救われている
- 狩人たちは彼の助言を重視する
- 子どもたちは彼を恐れている

**能力**:
- 危険生物の行動予測（繁殖期、移動ルート）
- 痕跡から獣の種類・数・状態を判別
- 狩人としての戦闘技術（最高レベル）
- 忍耐力（何日も観察小屋に籠もれる）

**物語での役割**:
- コウキの広域監視網が「なぜ獣が突然増えたのか」を明らかにする
- ダルガンの現地知識と組み合わさって危機への対策を発見
- 「データ（コウキ）vs 現場の知恵（ダルガン）」の融合
- 生態系の変化を物語の重要な要素として描く

**関係性**:
- 弟子：リク（18歳男性）とセナ（16歳女性）
- 妻：エルダ（集落に残っている、夫の仕事を理解しつつ心配）
- 協力者：近隣集落の狩人たちと情報交換

### 3. セリカ（古代を探る者）

**基本情報**:
- 年齢：20代女性
- 出身：都市遺跡周辺集落「旧都の門」（人口2,100人）
- 現在地：旧都の門
- 職業：研究職（記録係の補佐）

**外見**:
- 細身で華奢な体格
- 眼鏡をかけている（稀少品、ガラスレンズ）
- インクで汚れた指
- 好奇心に満ちた瞳

**性格**:
- 知的好奇心が旺盛
- 理想主義的で頑固
- 集落の保守的な空気に不満
- 「祖先の知識を取り戻す」という使命感

**背景**:
- 集落の「記録係」の娘として育つ
- 幼い頃から古代の遺物に囲まれて育った
- 「大災害前の技術を取り戻せば、もっと豊かに暮らせる」という信念
- 長老たちは「余計なことをするな」と彼女を抑制しようとする

**現在の活動**:
- 廃墟となった旧都市の遺跡を探索（週に1-2回、護衛付き）
- 古代の道具や文字（読めないが）を収集
- 隊商に依頼して、他の遺跡からの情報を集める
- 集落の長老会議に「古代技術の復元」を提案（却下されることが多い）
- 時折、彼女の発見が集落に利益をもたらす（新しい作物の栽培法など）

**口癖・名言**:
「私たちは祖先より貧しく生きている。それは間違っている」
「知識を失うことは、死ぬことより恐ろしい」

**周囲の評価**:
- 「理想主義の若者」と冷ややかに見られる
- しかし実績もあるため完全には否定されない
- 一部の若者は彼女に共感している
- 長老たちは「危険な思想」として警戒

**能力**:
- 古代文字の解読（一部のみ、試行錯誤）
- 遺物の機能推測（観察力と推理力）
- 記録・整理能力（几帳面）
- 説得力のある文章を書く

**物語での役割**:
- コウキと出会い、彼が「調査隊の一員」だと気づく（または疑う）
- 彼女の古代知識とコウキの技術が組み合わさって重要な発見
- 「過去を知る者（セリカ）vs 未来を知る者（コウキ）」の対比
- 真実を知ったときの葛藤（知識を公開すべきか、隠すべきか）

**関係性**:
- 父：記録係のエリウス（娘の探求心を理解しつつ心配）
- 協力者：若い護衛トーマス（遺跡探索に同行）
- 対立者：保守派の長老たち

### 4. オルド（放浪の語り部）

**基本情報**:
- 年齢：50代男性
- 出身：南部の大規模集落（既に忘れた）
- 現在地：放浪中（拠点なし）
- 職業：元隊商（30年以上）→ 現在は放浪の語り部

**外見**:
- やせ細った体格だが驚くほど頑健
- 日焼けと風雨で刻まれた深い皺
- 白髪混じりの長い髪
- 歩き続けた足の傷跡

**性格**:
- 穏やかで達観している
- 死を恐れない（むしろ受け入れている）
- 純粋な好奇心の塊
- 孤独を愛する

**背景**:
- 隊商として30年以上、ヤヒロ中を旅した
- 「世界の果て」を見たいという幼い頃からの夢
- 引退後も旅をやめられず、一人で放浪を続ける
- 妻と子どもはいたが、既に他界

**現在の活動**:
- 誰も行かない辺境を訪れる
- 各地で物語を語り、食事と宿を得る
- 「世界の果て（球殻の端）」に到達することが唯一の目標
- 時折、隊商に同行して遠方へ向かう

**口癖・名言**:
「この世界には必ず果てがある。誰も見たことがないものを、この目で見たい」
「死ぬ前に、もう一つだけ、見たことのないものを」

**周囲の評価**:
- 「哀れな老人」「頭がおかしくなった」
- しかし彼の語る物語は貴重な情報源
- 子どもたちは彼の冒険譚を楽しみにしている
- 集落の人々は彼に食事と宿を提供する（情報の対価）

**能力**:
- 膨大な地理知識（訪れた場所は数百箇所）
- 物語を語る技術（人を惹きつける）
- サバイバル能力（一人で辺境を旅できる）
- 危険察知能力（長年の経験）

**物語での役割**:
- コウキが出会う「もう一人の冒険者」
- ただし彼の冒険は「果てを見る」という個人的な夢
- 調査隊のような使命感はない
- 対比として機能（個人の夢 vs 人類の使命）
- 最終的に「世界の果て」の真実を知る展開も可能

**関係性**:
- かつての隊商仲間たち（各地に散らばっている）
- 孫のような若者たち（各地で出会う）
- 誰とも深い関係は築かない（別れが前提）

### 5. ミラ（医療探求者）

**基本情報**:
- 年齢：30代女性
- 出身：中規模集落「川辺の里」（人口500人）
- 現在地：複数の集落を巡回（数ヶ月ごとに移動）
- 職業：医療職（各地を巡る「旅する医者」）

**外見**:
- 清潔感のある身なり
- 常に薬草の香りが漂う
- 優しい表情だが疲労の色も
- 大きな薬草袋を背負っている

**性格**:
- 慈愛に満ちた性格
- 「治せない」ことへの強い焦燥感
- 完璧主義で自分に厳しい
- 患者のためなら危険も顧みない

**背景**:
- 医療職として優秀だったが、疫病で多くの患者を失った
- 「他の集落にはもっと良い治療法があるのでは」と考える
- 集落を出て、各地の医療知識を集める旅に出る
- 5年間で50以上の集落を訪問

**現在の活動**:
- 各地の「薬草師」「医療職」から治療法を学ぶ
- 薬草の標本を収集し、効能を記録
- 隊商の護衛と交渉し、危険地帯の珍しい薬草を採取してもらう
- 治療の成功例を記録し、知識を体系化しようとする
- 報酬：治療費（10-100穀/人）、宿と食事の提供

**口癖・名言**:
「目の前で人が死ぬのを見ているだけなんて耐えられない」
「どこかに答えがあるはず。諦めたらそこで終わり」

**周囲の評価**:
- 「命の恩人」として尊敬される
- 実利的で、周囲にも理解されやすい
- ただし「そこまでする必要があるのか」という疑問の声も
- 患者からは絶大な信頼

**能力**:
- 広範な医療知識（50集落分の治療法）
- 薬草の識別・調合技術
- 診断能力（症状から病気を推測）
- 忍耐力（困難な治療にも諦めない）

**物語での役割**:
- コウキの広域監視網が「新種の病気の発生源」を特定
- ミラの医療知識と組み合わさって治療法が見つかる
- 最も「主人公に協力的」なキャラクター
- 人間ドラマを描きやすい（患者との関係）

**関係性**:
- 師匠：故郷の老医師（既に他界、彼女の原点）
- 協力者：各地の医療職たち（ネットワークを構築している）
- 患者：数え切れないほど（彼女を慕う人々）

## 【キャラクター設計の原則】

### 1. 実利的な動機
- 純粋な冒険心だけでは周囲に受け入れられない
- 「集落/隊商/人々のため」という大義名分が必要

### 2. 変わり者としての立ち位置
- 尊重はされるが、完全には理解されない
- 「そこまでする必要があるのか」という疑問の目

### 3. 孤独な戦い
- 理解者は少ない
- しかし時折、彼らの知識が命を救う

### 4. コウキとの対比
- コウキ：調査隊の使命、広域監視網という技術
- 彼ら：個人的な動機、現地での生の経験
- 両者が協力することで、真の解決策が見つかる

### 5. 物語への貢献
- 世界の多様性と奥行きを表現
- 読者の共感を得やすい（コウキより人間的）
- 異なる視点を提供

## まとめ

これらのキャラクターは、ヤヒロという閉鎖的な世界において、極めて稀な「未知への探求心」を持つ人々です。彼らはそれぞれ異なる動機と方法で、世界を理解しようとしています。

コウキと出会うことで、彼らの知識と経験が、調査隊の技術と融合し、より大きな発見や問題解決につながります。

物語に登場させる際は、彼らの「変わり者」としての苦悩、周囲との軋轢、そして信念を貫く強さを描くことで、世界に深みと人間ドラマを加えることができます。
